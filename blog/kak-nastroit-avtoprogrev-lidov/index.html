Как настроить автопрогрев лидов: цепочка писем, SMS и чат-бот, которая работает без менеджера

Менеджер позвонил один раз, не дозвонился и забыл. А лид тем временем ушёл к конкуренту, который был настойчивее. Знакомая картина? Автопрогрев решает эту проблему — система сама касается лида 5-7 раз, пока он не будет готов к разговору.

Из чего состоит цепочка автопрогрева

  1. Мгновенная реакция (0 минут): лид оставил заявку → SMS/сообщение в мессенджер: «Спасибо, мы получили заявку. Менеджер позвонит в течение 15 минут»
  2. Первое касание (15 минут): звонок менеджера. Не дозвонился → автосообщение с предложением выбрать время для звонка
  3. Полезный контент (2 часа): кейс или статья по теме запроса лида. Без продажи — только польза
  4. Социальное доказательство (24 часа): отзыв клиента из той же ниши
  5. Спецпредложение (48 часов): «Только для вас — аудит бесплатно до пятницы»
  6. Финальное касание (72 часа): «Мы всё ещё здесь. Нужна помощь?»

Инструменты для настройки

  • Email: Unisender, Mailchimp — для цепочек писем
  • SMS: SMS-шлюзы через API
  • Чат-боты: Telegram, WhatsApp — для мгновенных касаний
  • CRM: AmoCRM — автоматическая постановка задач менеджеру
Результат: лид получает 5-7 касаний без участия человека. Менеджер подключается только когда лид «горячий» и готов к разговору.

Типичные ошибки

  • Слишком частые касания (спам)
  • Одинаковый контент на всех этапах
  • Нет возможности отписаться
  • Цепочка не привязана к источнику лида
