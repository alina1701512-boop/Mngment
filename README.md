# SystemSales — Маркетинговое агентство полного цикла

Сайт: [alina1701512-boop.github.io/Mngment](https://alina1701512-boop.github.io/Mngment/)

## О проекте

Статический сайт маркетингового агентства. Содержит 13 страниц услуг, 25 статей блога, 10 кейсов, 3 калькулятора, страницу контактов и лид-магниты.

## Технологии

- HTML5, CSS3, JavaScript (ванильный)
- GitHub Pages
- Микроразметка Schema.org (JSON-LD)
- Адаптивная вёрстка (mobile-first)

## Структура

- `/services/` — 13 страниц услуг
- `/blog/` — 25 статей + страница-хаб
- `/cases/` — 10 кейсов + страница-хаб
- `/tools/` — 3 калькулятора/теста
- `/contacts/` — контакты + бриф
- `/thank-you/` — страница благодарности
- `/404.html` — страница не найдена
